Python Library for SB Motor-shield

SB Components' Motor-Shield python library v.1.

Pi Motor-Shield Library Reference:

init()
This function initialises the motor shield and configures the GPIO as input/output for LED, sensor connectors and motor connectors.

arrow(position , state)
This function enables or disables the arrow LEDs.
Arguments
position: string – allowed values ‘FORWARD’, ‘BACKWARD’, ‘LEFT’ and ‘RIGHT’
state: string – allowed values ‘ON’ and ‘OFF’
example 

arrow('FORWARD', 'ON')

move(motor, mstate)
This function is used to select a motor/group and start or stop that motor/group. It also allows you to control the direction of the motor/group.
Arguments
motor: string – allowed values ‘MOTOR1’, ‘MOTOR2’, ‘MOTOR3’ and ‘MOTOR4’
mstate: string – allowed values ‘FORWARD’, ‘BACKWARD’ and ‘STOP’
example 

move('MOTOR', 'BACKWARD')

allstop()
This function stop all the motors. example

allstop()

readIR1() & readIR2()
These functions read the IR1 sensor and IR2 sensor inputs and return true if the sensors have been triggered otherwise the return false.

example 

Sensor_IR1 = readIR1()  
Sensor_IR2 = readIR2()
